## Digital Influencers API

## Install

````shell
docker run --rm \
    -u "$(id -u):$(id -g)" \
    -v "$(pwd):/var/www/html" \
    -w /var/www/html \
    laravelsail/php82-composer:latest \
    composer install --ignore-platform-reqs
    
  cp .env.example .env
  
  php artisan key:generate
  
  php artisan migrate --seed
````
